ffbatterymon(1)
===============

NAME
----
ffbatterymon - Fawkes battery monitor

SYNOPSIS
--------
[verse]
*ffbatterymon*

DESCRIPTION
-----------
This GUI application connects to robots and displays their current
battery charge level.
