ffnetloggui(1)
==============

NAME
----
ffnetloggui - Watch Fawkes log messages remotely

SYNOPSIS
--------
[verse]
*ffnetlogview*

DESCRIPTION
-----------
This GUI application allows for viewing log messages remotely. Once
started it will try to connect to Fawkes instances on the local
network (found via mDNS-SD).
